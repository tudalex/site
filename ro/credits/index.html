rosedu.org este un proiect open-source ce poate fi gasit pe GitHub sub o licenta CC-BY-SA sau GNU FDL 1.3 si mentinut de catre un grup de tineri entuziasti. Puteti contacta oricand echipa la adresa site@rosedu.org.


Tehnologii folosite

Jekyll

Jekyll este un generator static de pagini web. Pleaca de la un director sablon, il trece prin Textile sau Markdown si Liquid converters si produce pagini web complete, statice, potrivite pentru un server Apache sau serverul vostru web preferat.

Website: http://jekyllrb.com/

Licenta: MIT


liteAccordion

liteAccordion este un pugin orizontal de tip ‘acordeon’ pentru jQuery.

Website: http://nicolahibbert.com/demo/liteAccordion/

Licenta: MIT
